## markdown的简单介绍

Markdown是近几年兴起一种新型排版语言**规范**，在计算机相关行业十分流行。计算机相关专业的人，大二掌握基本的markdown用法，对于今后学习有很多便利性。

Markdown文件的后缀一般使用 `.md` ，如果你觉得这不好听，使用 `.markdown` 也可以，都属于文本文件。本文就是使用 markdown 编写的。
